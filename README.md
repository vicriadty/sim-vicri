## ‚öôÔ∏è Teknologi yang Digunakan

```python
# Contoh kode face recognition
import cv2
import face_recognition
import numpy as np

class FaceRecognitionQueue:
    def __init__(self):
        self.known_face_encodings = []
        self.known_face_names = []
        
    def add_customer(self, image, name):
        # Tambah wajah baru ke database
        face_encoding = face_recognition.face_encodings(image)[0]
        self.known_face_encodings.append(face_encoding)
        self.known_face_names.append(name)
```

## üíæ Database Schema

### Tabel Pengunjung
```sql
CREATE TABLE visitors (
    id INT PRIMARY KEY AUTO_INCREMENT,
    face_encoding BLOB,
    name VARCHAR(100),
    registration_date DATETIME,
    last_visit DATETIME,
    total_visits INT DEFAULT 0
);
```

### Tabel Antrian
```sql
CREATE TABLE queue (
    queue_id INT PRIMARY KEY AUTO_INCREMENT,
    visitor_id INT,
    assigned_counter INT,
    queue_number VARCHAR(10),
    status ENUM('waiting', 'serving', 'completed'),
    created_at DATETIME,
    served_at DATETIME
);
```
